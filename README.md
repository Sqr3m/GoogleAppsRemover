# Google Apps Remover

## Instruction

- Install `git`, `adb`

- ```git clone https://github.com/Sqr3m/GoogleAppsRemover```

- Go to settings, find **"About phone"**, click 7 times on **Build number**, then find **"Developer options"** and go there, find the **DEBUGGING** section and finally enable **USB Debugging**

- Connect your phone via cable. Wait. **Allow USB debugging *(OK)***

- ```sh remover.sh```

---

## Troubleshoot

### *If dialog doesn\'t appear try to*

- *Reconnect phone*
- *Disable/enable **USB Debugging***
- *Reboot phone*
