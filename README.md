## Instruction

1. Install `git`, `adb`

2. ```git clone https://github.com/Sqr3m/GoogleAppsRemover```

3. Go to settings, find **"About phone"**, click 7 times on **Build number**, then find **"Developer options"** and go there, find the **DEBUGGING** section and finally enable **USB Debugging**

4. Connect your phone via cable. Wait. **Allow USB debugging *(OK)***

5. ```sh remover.sh```

6. Or download binary in **Releases**

7. ```chmod +x Remover && ./Remover```

---

## Troubleshoot

### *If dialog doesn\'t appear try to*

- *Reconnect phone*
- *Disable/enable **USB Debugging***
- *Reboot phone*
- *Type **adb + `space`** in the console, then **`enter`***
